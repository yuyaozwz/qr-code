# Java二维码批量生成工具

### 介绍
基于Java的二维码生成工具，支持自定义二维码内容、内嵌图片、批量生成！

### 二维码生成样例    
该二维码的内容为 https://zwz99.blog.csdn.net/，即我的博客，扫码后将会打开此网站。    
二维码本质上就是一个字符串，扫码后将显示这个字符串。    
如果这个字符串是网址，将会被直接打开。    
如果这个网址被第三方平台所绑定（如微信扫码微信平台绑定的网址），将会跳转到指定页面（如微信小程序）
